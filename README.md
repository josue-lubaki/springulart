## dependencies
- spring-boot-starter-security
- spring-boot-starter-mail 
- S3 Amazon (Storage)
- guava (google)
- java-jwt (Auth0)
- springfox-swagger2

- spring-boot-starter-data-jpa
- thymeleaf
- postgresql
- Liquibase (Migration)
